Trabalhando com JSON com a Biblioteca GSON

JSON

    JSON (JavaScript Object Notation), formato para armazenamento e transmisso de dados, assim como o XML. Tem uma legibilidade fcil para nos humanos e simples para o computador gerar e fazer o parser. O JSON menos verboso que um XML.
    Quanto a transmisso de dados, como o JSON texto, ele independente de linguagem de programao, logo podemos utilizar ele para fazer programas feitos em linguagens diferentes "conversarem".
    Para mais informaes acesse o site oficial http://www.json.org.

Exemplos de Estruturas em JSON

Exemplo de Objeto (cliente) em JSON

{
	id : 1,
	nome : "Victor",
	email : "victormenegusso@gmail.com"
}
  • "{" indica o inicio do objeto.
  • "}" indica o final do objeto.
  • Dentro de um objeto temos atributos, que so formados por nome/valor, separados pelo caracter ":".

Exemplo de Lista de String (emails) em JSON

[ "victormenegusso@gmail.com" , "teste@teste.com" , "javeiro@php.com" ]
  • "[" indica o inicio de um array.
  • "]" indica o final de um array.
  • "," separa cada item da lista(array).

Exemplo de Lista de Objetos (lista de clientes) em JSON

[
	{
		id : 1,
		nome : "Victor",
		email : "victormenegusso@gmail.com"
	} ,
	
	{
		id : 2,
		nome : "Stalone",
		email : "stalone@mercenario.com"
	}
]

GSON

     uma Biblioteca Java, utilizada para converter Objetos Java em uma representao JSON (texto) e vice versa.
    Site: https://github.com/google/gson
    Manual: https://github.com/google/gson/blob/master/UserGuide.md
    Download Jar: gson-2.7.jar

Objeto JSON -> Objeto Java

    Quando vamos converter  um Objeto JSON para um Objeto em Java, devemos criar a classe em Java que represente este objeto, com os mesmos atributos definido no Json. 
    JSON:

Exemplo de Objeto (cliente) em JSON

{
	id : 1,
	nome : "Victor",
	email : "victormenegusso@gmail.com"
}
    
    Java:

Cliente em Java

package tecpuc.aula.json.gson;

public class Cliente {
	private int id;
	private String nome;
	private String email;
	
	public int getId() {
		return id;
	}
	public void setId(int id) {
		this.id = id;
	}
	public String getNome() {
		return nome;
	}
	public void setNome(String nome) {
		this.nome = nome;
	}
	public String getEmail() {
		return email;
	}
	public void setEmail(String email) {
		this.email = email;
	}
}
  • Cada atributo da Classe, deve ter o mesmo nome que o atributo no JSON.

Convertendo os dados JSON -> Java

Exemplo de converso Json -> Java

package tecpuc.aula.json.gson;

import com.google.gson.Gson;

public class JsonParaJavaObject {

	public static void main(String[] args) {
		
		// --- texto json --- //
		String json = "{\"id\":1,\"nome\":\"Victor\",\"email\":\"victormenegusso@gmail.com\"}";
		
		// --- transformando em Objeto Java --- //
		Gson gson = new Gson(); // conversor
		Cliente objCliente = gson.fromJson(json, Cliente.class);
		
		// exibindo dados em Java //
		System.out.println( objCliente.getId() );
		System.out.println( objCliente.getNome() );
		System.out.println( objCliente.getEmail() );
	}

}
  • Utilizamos o GSON para fazer as converses.
  • 'fromJson' o mtodo utilizado para transformar texto json para objeto Java, nele temos os parmetros:
    • 'json' -> texto a ser convertido.
    • 'Cliente.class' -> informamos a classe que deve ser utilizada na converso
  • 'imports' importantes:
    • 'import com.google.gson.Gson;' classe responsvel por fazer o parser.

Convertendo os dados Java -> Json

Java -> Json

package tecpuc.aula.json.gson;

import com.google.gson.Gson;

public class JavaObjectParaJson {

	public static void main(String[] args) {
		
		// --- criar objeto cliente --- //
		Cliente objCliente = new Cliente();
		
		objCliente.setId(1);
		objCliente.setNome("Victor");
		objCliente.setEmail("victormenegusso@gmail.com");
		
		// --- transformando em JSON --- //
		Gson gson = new Gson(); // conversor
		String json = gson.toJson( objCliente );
		
		// exibindo o JSON //
		System.out.println( json );

	}

}
  • Utilizamos o GSON para fazer as converses.
  • 'toJson' o mtodo utilizado para transformar objeto java em texto json, nele temos o parmetro:
    • 'objCliente' -> objeto a ser convertido.
  • 'imports' importantes:
    • 'import com.google.gson.Gson;' classe responsvel por fazer o parser.

    JSON:

Exemplo de Objeto (cliente) em JSON

[ "victormenegusso@gmail.com" , "teste@teste.com" , "javeiro@php.com" ]
    
    Java:
    No precisamos criar uma classe especial, pois lista utilizaremos o 'ArrayList' de 'String'.

Convertendo os dados JSON -> Java

Exemplo de converso Json -> Java

package tecpuc.aula.json.gson;

import java.lang.reflect.Type;
import java.util.ArrayList;

import com.google.gson.Gson;
import com.google.gson.reflect.TypeToken;

public class JsonParaJavaListString {
	public static void main(String[] args) {
		
		// json //
		String json = "[\"victormenegusso@gmail.com\",\"teste@teste.com\",\"javeiro@php.com\"]";
		
		// --- convertendo para json --- //
		Gson gson = new Gson();
		
		Type tipoLista = new TypeToken<ArrayList<String>>() {}.getType();
		ArrayList<String> lista = gson.fromJson(json, tipoLista);
		
		// --- exibindo json --- //
		for(String item : lista)
		{
			System.out.println(item);
		}
	}
}
  • Utilizamos o Gson para fazer as converses.
  • Utilizamos o 'Type' e 'TypeToken' para indicar o tipo, pois temos a implementao da lista mais o tipo de objeto.
  • A varivel 'tipoLista' armazena informaes sobre o tipo da lista, vamos ter que fazer esse procedimento sempre que utilizarmos lista. 
  • 'fromJson' o mtodo utilizado para transformar texto json para objeto Java, nele temos os parmetros:
    • 'json' -> texto a ser convertido.
    • 'tipoLista' -> informamos o tipo de dados que deve ser utilizada na converso
  • 'imports' importantes:
    • 'import com.google.gson.Gson;' classe responsvel por fazer o parser.
    • 'import java.lang.reflect.Type;'
    • 'import com.google.gson.reflect.TypeToken;'

Convertendo os dados Java -> Json

Java -> Json

package tecpuc.aula.json.gson;

import java.util.ArrayList;

import com.google.gson.Gson;

public class JavaListStringParaJson {
	public static void main(String[] args) {
		
		// lista de texto //
		ArrayList<String> lista = new ArrayList<String>();
		lista.add("victormenegusso@gmail.com");
		lista.add("teste@teste.com");
		lista.add("javeiro@php.com");
		
		// --- convertendo para json --- //
		Gson gson = new Gson();
		String json = gson.toJson(lista);
		
		// --- exibindo json --- //
		System.out.println(json);
	}
}

Convertendo os dados Java -> Json

Java -> Json

package tecpuc.aula.json.gson;

import java.util.ArrayList;

import com.google.gson.Gson;

public class JavaListClienteParaJson {
	public static void main(String[] args) {
		
		// lista de cliente //
		ArrayList<Cliente> lista = new ArrayList<Cliente>();
		Cliente novoCliente;
		
		novoCliente = new Cliente();
		novoCliente.setId(1);
		novoCliente.setNome("Victor");
		novoCliente.setEmail("victormenegusso@gmail.com");
		lista.add(novoCliente);
		
		novoCliente = new Cliente();
		novoCliente.setId(2);
		novoCliente.setNome("Stalone");
		novoCliente.setEmail("stalone@mercenario.com");
		lista.add(novoCliente);
		
		// --- convertendo para json --- //
		Gson gson = new Gson();
		String json = gson.toJson(lista);
		
		// --- exibindo json --- //
		System.out.println(json);
	}
}
